# Spesifikasi Teknis Meja Kursi Sekolah

Repository ini berisi data spesifikasi teknis untuk pengadaan furnitur sekolah standar nasional.

## Tentang Project
Data ini digunakan untuk acuan standar ukuran meja siswa SD, SMP, SMA, dan Kampus.
